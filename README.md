## üí° Installation /Instalaci√≥n
***

**Installation of the Backend Laravel Project**
 1. Clone the Laravel repository from GitHut
 `https://github.com/aura-souls/colores-de-la-luz.git`
 2. Open the project in your code editor and in the terminal enter the project directory
 `cd back`
 3. Installs the Composer dependencies
 `composer install`
 4. Creates a . env file from the .env.example file.
 5. Create your database in mysql with the name of your choice.
 6. Configure the database in the file . env with the appropriate information, where you must put the name of your database previously created in mysql and be clear about your operating system for the installation.Example:
 <pre>
<code>
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=nombre_base_de_datos
DB_USERNAME=nombre_usuario
DB_PASSWORD=contrase√±a
</code>
</pre>
7. Run migrations to create database tables
`php artisan migrate`
8. Start the development server
`php artisan serve`

**Installation of the Frontend React Project:**

1. Enter the project directory
 `cd front`
2. Install npm dependencies
`npm install`
3. To get the API
`npm install axios`
4. To install React testing Library 
`npm install --save-dev @testing-library/react`
5. To install React-router-dom
`npm install --save react-router-dom`
6. Start the project
`npm start`
7. React frontend is now up and running on 
`http://localhost:3000`
